The big integer multiplication algorithm multiplies two given large integers.

Input: Two hexadecimal byte arrays \( firstFactor, secondFactor \) of dimension n1 and n2
Output: Hexadecimal byte array \( result \)
Test of correctness:     The resulting array \( result \) is correct iff arrays \( firstFactor \) and \( secondFactor \) converted to decimal numbers hold the following equation: $$ firstFactor_{10} * secondFactor_{10} = result_{10} $$
Implementation: To implement a big integer multipliaction algorithm the author has to write a java class extending the BigIntegerMultiplicationAbsAlgorithm class and implementing its execute(byte [] firstFactor, byte [] secondFactor, byte [] result) method.
